Maine Opportunity Zone Eligibility Map
About the Tool
This tool allows users to visually explore which census tracts in Maine are eligible for Opportunity Zone nomination under the provisions of the 2025 One Big Beautiful Bill Act (OBBBA). Users can search for a specific address to see which census tract it falls within and view tract-level demographic and economic data used to determine eligibility.
Features
Interactive census tract polygons color-coded by OZ eligibility status. Address search to locate a specific address on the map. Clickable popups displaying tract-level data including poverty rate, median family income, unemployment rate, and population. Town and county boundary overlays for geographic reference. Toggleable town labels via the layer control panel.
Data Sources
Census tract eligibility criteria: Based on the Opportunity Zone provisions established in the 2025 One Big Beautiful Bill Act (OBBBA). Demographic data: Independently reconstructed and verified by the Maine Office of the State Economist, Department of Administrative and Financial Services from the 2020–2024 American Community Survey 5-Year Estimates (U.S. Census Bureau), cross-referenced with data published by the Economic Innovation Group. Geographic boundaries: U.S. Census Bureau TIGER/Line Shapefiles.
Built With
R, leaflet (interactive map rendering), leaflet.extras (address search functionality), sf (spatial data processing).
Notes
This map is intended for informational purposes only. Users should verify eligibility status through official sources before making programmatic decisions.
