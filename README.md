# 2026 Maine Opportunity Zone Tract Eligibility Map

## About the Tool
This tool allows users to visually explore which census tracts in Maine are eligible for the 2026 Opportunity Zone nomination cycle under the provisions of the 2025 One Big Beautiful Bill Act (OBBBA). Users can search for a specific address to see which census tract it falls within and view tract-level eligibility status. Prepared by the Maine Office of the State Economist.


## Features
Interactive census tract polygons color-coded by OZ eligibility status. Address search to locate a specific address on the map. Clickable popups displaying census tract eligibility status. Town and county boundary overlays for geographic reference. Toggleable town labels via the layer control panel. 

## Data Sources
Census tract eligibility is determined by criteria established in the 2025 One Big Beautiful Bill Act (OBBBA) for the 2026 Opportunity Zone nomination cycle. The Maine Office of the State Economist independently interpreted the OBBBA eligibility provisions, collected the required 2020–2024 American Community Survey 5-Year Estimates data at the census tract level, and calculated eligibility determinations for all Maine tracts. Results were validated against eligibility data published by the Economic Innovation Group. Geographic boundaries: U.S. Census Bureau TIGER/Line Shapefiles.

## Built With
R, leaflet (interactive map rendering), leaflet.extras (address search functionality), sf (spatial data processing).
